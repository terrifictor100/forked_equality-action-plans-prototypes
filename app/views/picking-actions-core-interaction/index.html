Picking actions - core interaction options

Key challenge

Each action that might be chosen has a substantial amount of supporting information - at least a paragraph of text. Some users will want to read the supporting information for each action, while some will already know what they want to pick and just want quick access to selecting their actions.

These visualisations show different ways of displaying that information to provide for both use cases.
